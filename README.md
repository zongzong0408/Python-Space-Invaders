- Windows Home 10 64bit
- VScode
- Python `3.7.0`
- pygame

---

## 如何執行

> 預設 ***Windows Home 10 64bit*** 作業系統  

步驟 一：下載 `pygame` 。  

步驟 二：執行。  

- 方法一：直接點兩下執行 `analysis.py`  
>
- 方法二： 

```bash
cd Python-space-invaders
```
```bash
python game.py
```
